# Tech Logos

This folder contains all technology logos used in the portfolio.

## How to Add Logos

1. Download logos from [Devicon](https://devicon.dev/) or official sources
2. Save as SVG or PNG format
3. Name them clearly (e.g., `angular.svg`, `react.svg`)
4. Reference them in `Skills.tsx` component
